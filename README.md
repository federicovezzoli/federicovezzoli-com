# federicovezzoli.com

Personal website for [Federico Vezzoli](https://federicovezzoli.com), built with Nuxt.js, deployed as a static site on Netlify.

## Tech Stack

- **[Nuxt.js 2](https://nuxtjs.org)** — static site generation
- **[GSAP](https://greensock.com/gsap/)** — animations
- **[Netlify](https://netlify.com)** — hosting and deployment
- **SCSS** — styling with Bootstrap utilities

## Getting Started

```bash
# install dependencies
npm install

# serve with hot reload at localhost:3000
npm run dev
```

## Build & Deploy

```bash
# build for production and launch server
npm run build
npm start

# generate static output
npm run generate
```

Deployments are handled automatically by Netlify on push to the main branch.

## Lint

```bash
npm run lint
```
